Welcome to Fandom Stats!

We are working on a set of tools that will make it easier to get data for fandom analysis, mostly from the popular and number-friendly fandom haunts like AO3 or tumblr.

On this page you can try out an early version of a tool like that - search for any AO3 tag, and it will create visualisations about it, using our AO3 Tag Stats API to gather the data. Have fun!

Where to go

  • Try the AO3 Bookmark Viewer
  • Learn more about the Tag Stats API, and what is it for
  • Read about the people behind this
  • Some caveats and notes about the data and the methods

Get stats for AO3 tags

You can search for a tag or a combination of tags - in that case, use double quotes around tags with spaces. To better understand what the graphs are saying, check out "Reading the Data".
