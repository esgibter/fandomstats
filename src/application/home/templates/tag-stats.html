AO3 Tag Stats Tool

The purpose of this tool is to make it easy to fetch and display basic stats about any type of Archive Of Our Own (AO3) tag. For example, let’s say you want to know more about how any of the following types of AO3 tags are used:

  • A fandom tag: “Teen Wolf (TV)”
  • A character tag: “Rey (Star Wars)”
  • A relationship tag: “Fox Mulder/Dana Scully”
  • A rating tag: “Mature”
  • A relationship category tag: “F/F”
  • A warning tag: “Graphic Depictions Of Violence”
  • A freeform (additional) tag: “Angst”

… and we return to you data about how that tag is used on AO3, in JSON format.

Wait, what is this?!

Right now, we’re simply returning to you the same numbers that you’d see in the AO3 Sort & Filter sidebar if you clicked on the tag on AO3. This is a bit of a proof of concept and placeholder for what we hope to do in the future. As for why it’s in JSON, see more below.

What will it become?

We have three goals:

  1. Display graphs on this page. We want to start displaying some graphs of this data here, so that people who never want to write any programs or do any stats of their own can just type in an AO3 tag and get back some nice quick data visualization of how the tag is used.
  2. Provide an AO3 fandom stats API. The reason the tag data is displayed in JSON format is that we’re also trying to write a more general purpose set of Python functions that fandom data geeks can call from within their own programs in order to efficiently fetch data about tags from AO3. We’d like to also eventually provide more data aside from that shown here (e.g., the ability to fetch median hits and median kudos for the tag.)
  3. Provide other fandom data-related information and tools elsewhere on the site. Right now, you can see our AO3 Bookmark Viewer as one such example.

How can I use the existing API -- or help develop it further?

If you want to use our existing Python scripts, or help develop the API or the site further, please contact us.
