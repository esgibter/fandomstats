Resources: AO3 daily activity data

Author(s): AO3/OTW

Want to know how many works are on AO3, or how many are added per week? Here are some data for you to play with yourself.

The data files are comma separated lists of activity per day on AO3 as assessed on the date in the file name (20170415 is the 15th of April, 2017). The counts in these files are listed in reverse chronological order.

The following two files are counts of the works currently available on AO3 on the dates of assessment. Each row consists of: date, count with the date reported as an integer with YYYYMMDD.

  • AO3_worksPosted_20170415.csv
  • AO3_worksPosted_20170924.csv

Also available are the number of works being updated per day. Each row consists of: days prior to assessment, count with days starting from 1.

  • AO3_Updated_20170930.csv

And the number of chapters added per day, Each row is consists of: days prior to assessment, count with days starting from 1.

  • AO3_Chapters_20170930.csv
